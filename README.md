# Лабараторная 3-4 прога
## В UML диаграмме:

- participants   - все участники, кроме пленников
- prisoners      - отдельный пул пленников

Компиляция диаграммы в svg:
```
java -jar plantuml.jar -tsvg diagram.puml
```

Для диаграммы использовался PlantUML
Cкачать его можно тут: https://plantuml.com/ru/download
